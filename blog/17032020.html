Smallest Dev Environment?

Quarantine Experiment: What is the smallest dev environment I can live with?

By having a smaller set of tools, you can focus on mastering them in a shorter time. Additionally, it is easier to keep track of versions and vulnerabilities.
I will use Alpine Linux and a browser. In addition I will run the following: apk update apk add curl git jq lastpass-cli libressl nodejs npm tree tmux
